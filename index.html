Selected Projects

Python FastAPI TensorFlow BiLSTM WebSocket

Real-Time Sign Language Recognition System

A thesis project that performs real-time Filipino Sign Language recognition using a Bidirectional LSTM model with Attention Pooling. Streams live hand landmark sequences via WebSocket to a FastAPI backend, which classifies gestures using a trained BiLSTM-Attention model and returns predictions with confidence scores in real time.

Live Demo GitHub (Frontend) GitHub (Backend)
Python Streamlit SQLite WebSocket FastAPI

Office Reminder System

A LAN-based office reminder web app built with Streamlit. Allows staff to send, schedule, and track reminders across devices on the same network. Features real-time toast notifications, receipt upload & verification workflow, snooze/escalation chains, push notifications via Web Push API, and a full activity log — all powered by a local SQLite database and a FastAPI notification bridge.

Demo (Drive) GitHub
HTML CSS JavaScript Python Selenium PHP

ERC Document Management System

A web-based document management system for browsing and searching official issuances — Resolutions, Orders, and Decisions — from the Energy Regulatory Commission (ERC) of the Philippines. Powered by a Python + Selenium web scraper that extracts documents directly from erc.gov.ph, with filtering by type and year, pagination, and a PHP backend for data persistence.

Demo (Drive) GitHub
C# .NET 6 Windows Forms EPPlus Excel

ZAMECO I Data Entry System

A Windows desktop application built for Zambales Electric Cooperative I (ZAMECO I) to streamline internal data entry workflows. Built with C# and .NET 6 Windows Forms, it uses EPPlus to read, write, and export structured data directly to Excel — replacing manual spreadsheet work with a clean, form-based interface.

Demo (Drive) GitHub
HTML CSS JavaScript Role-Based Access CSV Export

DOE DEPDMS Monthly Submission Form

A web-based data management system for the Department of Energy (DOE) Philippines. Features role-based login per department — Finance, Technical Services, and Power Trading — with each department accessing their own set of monthly submission forms. Includes multi-tab form navigation, locked sections per role, and admin-level CSV export for consolidated reporting.
